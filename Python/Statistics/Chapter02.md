### 2.2.2. 사분위 범위
> 데이터 하위 25%, 50%, 75%에 위치하는 값으로 범위 분할<br/>
> 중앙값에 대해서 정의되는 산포도의 지표<br/>
> 25% = 제1사분위수 = Q1<br/>
> 50% = 제2사분위수 = Q2 = 중앙값<br/>
> 75% = 제3사분위수 = Q3<br/>
> 사분위 범위(IQR): $IQR = Q3 - Q1$<br/>
- NumPy 라이브러리 활용
```
import numpy as np
Q1 = np.percentile(scores, 25)
Q1 = np.percentile(scores, 25)
IQR = Q3 - Q1
```
